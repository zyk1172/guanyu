# 观隅浏览器助手

Manifest V3 浏览器插件 MVP，用于把当前新闻页面或选中文本发送到观隅 Web App。

## 功能

- 分析当前页面
- 分析选中文本
- 保存到观隅
- 打开观隅 Web App
- 使用插件专用 token 绑定账号

当前版本已完成账号绑定、token 校验、页面正文提取和发送框架。`save` 可读取页面并返回观隅入口；`analyze` 需要后端继续把 `/api/analyze` 抽成共享生成服务后正式开放。

## 安装开发版

1. 打开 Chrome 扩展程序页面。
2. 开启“开发者模式”。
3. 选择“加载已解压的扩展程序”。
4. 选择本目录 `extension/`。

## 账号绑定

1. 登录观隅 Web App。
2. 打开 `/account/extensions`。
3. 点击“生成插件连接码”。
4. 在插件弹窗中输入连接码。
5. 插件换取 `extensionToken` 后保存到 `chrome.storage.local`。

## 权限说明

插件权限保持克制：

- `activeTab`：仅在用户主动点击时读取当前标签页。
- `scripting`：按需注入正文提取脚本。
- `contextMenus`：右键菜单入口。
- `storage`：保存插件专用 token 和展示用用户缓存。

插件不申请：

- 浏览历史权限
- Cookie 权限
- webRequest 权限
- 数据库访问权限

## 隐私说明

1. 插件只在用户主动点击“分析当前页面”“分析选中文本”“保存到观隅”时读取页面内容。
2. 插件不会后台监控浏览记录。
3. 插件不会自动上传所有网页。
4. 插件不会读取或保存用户密码。
5. 插件不会读取或保存用户模型 API Token。
6. 插件提交的内容会发送到观隅 Web App，用于后续生成新闻叙事审视报告。
7. 用户可以随时在账号页面撤销插件授权。

## 发布前配置

`manifest.json` 当前包含：

- `https://guanyu-seven.vercel.app/*`
- `http://localhost:3000/*`

正式发布前应移除本地开发域名，并只保留正式观隅域名。
